Conference on Scalability 的影片

上次說到 6/23 在 Seattle 有個 Conference on Scalability，昨天發現這些影片早就已經上線了。對於跟我一樣沒有參加 conference 卻對主題內容有興趣的人來說，可真是方便啊…

我把影片跟各個 session 的順序整理一下，一來自己需要，二來有興趣的人也可以直接取用 🙂

  • Keynote I : Google 的 Jeff Dean 講 Map/Reduce、BigTable、和 handling large datasets 的問題
  • Keynote II: Google 的 Marissa Mayer (first female engineer in Google) 談 Scaling Google for Every User
  • Cluster File Systems 的創辦人 Peter Braam 談 Lustre FS，linux 下的 cluster FS
  • University of British Columbia 的 Brad Penoff 等人談 SCTP 怎麼改善 Reliability 和 Fault tolerence
  • Symantec 的 Ryan Gerard 講 Scalable Test Selection Using Source Code Deltas
  • VeriSign 談他們的 Global DNS 架構
  • Google 的 Barry Brummit 談 Map/Reduce 怎麼用在大規模 geographic 資料上面
  • Amazon 的人談建立無限擴充的 datastore 碰到的瓶頸和困難 (想必是 S3 的前身) – 這個影片找不到？超可惜的說 >_
  • YouTube 的 Cuong Do 談 YouTube 怎麼撐到目前的 scale

Bon appetite.
