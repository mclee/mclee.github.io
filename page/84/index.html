不知道有沒有人對於替 File System 取名字有興趣…

自己的老婆被拘捕, 沒想到幾個月不到就傳出求售公司籌錢打
官司的新聞。

把 ReiserFS 改個名字, 而且 ReiserFS 的 popularity 也不低呢。
買下這家公司的錢, 會不會比捐錢蓋大樓然後替大樓命名划算
一點? :p

Mark Cuban & Donald Trump

剛剛在 Mark Cuban 的 Blog 上看到一篇很有趣的文章:
Trump the Chump.

文章一開始說, 很多人最近都在問他, 對於 Donald Trump
嗆他的事情, 到底有什麼感覺 :p (可以參考這裡、這裡和這裡。)

看起來兩個人結怨已經不是一天兩天的事情了, 可是 Mark
Cuban 巴 Donald Trump 可還是一點都不小力啊 😛 雖然 Trump
明顯把注意力轉移到 Rosie Donnel 身上了, 不過 Mark Cuban
還是抓住機會狠狠嘲笑了一翻 :p

除了嘲笑 Donald Trump 辦了 Trump University 只是為了騙錢,
讓一堆人為了 “成為有錢人” 這個夢想跑去買他的書、上他的課程,
然後替他工作以外, 也針對他的 blog 狠狠刮了一頓。:P 簡單的說
就是一句話:

「你這傢伙到底懂不懂 blog 啊? 根本沒有人要看你的 blog, 人家
也不 link 你的 blog, 你這傢伙根本是廢柴嘛，連新媒體和舊媒體都
分不清楚。讓我教你寫 blog 該是怎麼一回事！」

對看熱鬧的人來說, 是很有趣的筆戰文章 😛 對 blogger 來說, 是
很有意義的文章 – Mark Cuban 精確的抓住了一般 blogger 的心理。
更深一層來說, 是很有網路行銷教育意義的文章。Internet 和傳統
的媒體, 性質上是差很多的。但是即使是知名網路公司的員工, 還是
有許多人分不清楚。

有興趣的話, 這篇文章在這裡: Trump the Chump.

memcached 加速技巧

有人說過 memcached 對大型 object 的存取效率並不好。我自己沒有
用過, 也不知道這種說法真實度有多高…

But anyway, 這邊有人提出解法的小撇步, 把 memcached 的
TCP_NOPUSH undefine 掉,可以讓大型物件的存取速度變更快。
也許應該找個時間挖進去看看, 到底 define 了 TCP_NOPUSH
做了些什麼事情…
