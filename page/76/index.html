MySQL going to public ?

有則新聞是在說，在採訪 MySQL CEO 時，他透露出 MySQL 有要公開上市的計畫。聲稱雖然募集來的創投資金多數尚未動用，不過已經有在進行 IPO 的計畫，甚至可能在年底之前就準備上市。

聽起來好像是不錯的消息，可是 MySQL 雖然廣受 open source 歡迎，但是目前的 customer 數目只有大約一萬個，這樣產生的收益足夠作為 IPO 的題材嗎…?

CDBurnerXP and Multibyte

上個月看到 Channel-M 的 CDBurnerXP 介紹，看起來介面頗為美觀大方，而且又是 freeware 的 cdburner，就在自己的 notebook 上裝了一份。

最近剛好有需要備份檔案 (*嗶*)，準備把一些資料燒出來的時候，才發現「耶？怎麼我的那些中文檔名都變亂碼了呢？」原本以為是選錯了 cd 格式，可是回頭一瞧，明明是 Joliet 沒錯啊? :p

搞半天翻了一下才知道，原來在 FAQ 裡面就有寫，「我們目前的 burning library 不支援 multibyte character。等以後的版本看會不會改善吧。」

hmm, 這叫我該如何是好呢… 照理說似乎新版應該就會解決了，可是目前的 3.0.116 版好像是 2004 年七月出的了，以這個進度來說好像不太夠 active，叫我等下去我也怕從此沒消息了啊~~

看這個列表似乎是有很多可用的替代品，不過懶人如我還是等強者推薦以後我再找來用好了 Orz

Flash Player Penetration by Version

剛剛亂翻資料的時候，無意間看到這個：Flash Player: Version Penetration，是 Adobe 分別在 Jun 2006 和 Sep 2006 做的統計。

對於 Flash Player 版本市佔率有興趣的人可以看看…
幾個感想：

  1. flv 果然跨平台威力十足，幾乎大部分的使用者都支援。
  2. 是不是新興市場對於 Adobe 貢獻度不夠，連統計都懶得統計？:p
  3. 就算扣除新興市場，樣本國家數還是有點少，可惜了一點…
